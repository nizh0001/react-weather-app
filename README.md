# React Weather App

A weather application built with React and Vite. Search for locations and view real-time weather information.

## Features

- Search for locations by city, province, or country
- Save up to 5 locations
- View detailed weather information
- Clean and modern UI

## Setup

1. Clone the repository and install dependencies:
```bash
npm install
```

2. Get your API key from [OpenWeatherMap](https://openweathermap.org/api) (free account)

3. Create a `.env` file in the root directory:
```bash
cp .env.example .env
```

4. Add your API key to the `.env` file:
```
VITE_WEATHER_API_KEY=your_api_key_here
```

5. Run the development server:
```bash
npm run dev
```
